## ğŸ“š Dictionary for Developers

*   **Guard:** A class that implements the `CanActivate` interface and is used to protect routes.
*   **`CanActivate`:** An interface that a Guard must implement. It has a single method, `canActivate()`, which returns a boolean or a Promise/Observable of a boolean.
*   **`AuthGuard`:** A built-in Guard provided by `@nestjs/passport` that automatically invokes the appropriate Passport strategy.
